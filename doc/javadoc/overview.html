This document is the API specification for the tec json web application.

in order to use this application you will need the following third party applications:

  • JDK 8
  • Apache Tomcat (7)
  • Sencha Extjs (4.2.x)
  • Mysql 5.5

This application uses the following java third party libraries controlled via ivy:

  • java API
  • j2ee servlet API
  • Apache commons lang
  • Apache commons logging
  • JSON lib
  • Spring Framework
  • Spring Security
  • Hibernate
  • HikariCP Connection pool

The following are build and infrastructure tools used for this project:

  • Eclipse IDE for Java EE Developers
  • Ivy eclipse plugin
  • extjs jsdt plugin
  • Checkstyle eclipse plugin
  • Apache ANT
  • ANT contrib
  • Apache IVY
  • dbdeploy
  • Checkstyle
  • Sench CMD build tool (4.x)
  • ruby windows installer (pre 3.0)
